Laminas API Tools is the open source API builder for PHP, designed to simplify creating and maintaining useful, easy to consume, and well structured APIs. Regardless of your experience in API building, with Laminas API Tools you can build APIs that enable mobile apps, developer communities, and any other consumer controlled access to your applications.

Key features: RESTful or RPC services; JSON (specifically, HAL); Problem Details for HTTP APIs; Versioning; Normalisation and Validation; Authentication (HTTP Basic/Digest, OAuth2); Documentation (HTML, Swagger, and API Blueprint).

Laminas API Tools is made using Laminas, AngularJS and Bootstrap.

If you want to contribute, you can fork any of the relevant repositories under the laminascampus organization on GitHub.

The official web site of the project is api-tools.getlaminas.org.
